🛠️ How to Contribute

Contributions are welcome! To add or improve study materials:

Fork this repository

Create a new branch (git checkout -b feature/subject)

Add your files/content in a structured format

Commit your changes (git commit -m "Added [subject] resources")

Push to your fork

Open a Pull Request

Please make sure:

File names are clear and descriptive

No copyrighted or unsafe materials are added

You add a short summary of what each file contains
